1. You will need a sql host service like WampServer.

3. Create a database name "hellocine".
4. Go to the file "sql".
5. Upload "hellocine.sql" on the new database.
6. Go to tje file "JAR file".
7. Be sure to have a internet connection.
8. Run "HelloCine.jar".
9. Have fun!
